I knew software was my calling early — I just took the scenic route through a health-science program first. Three years into **Biochemistry**, I pivoted back to what I actually wanted to build: **Web3, DeFi, and systems that ship**.

Today I juggle both worlds. Biochemistry by day (or semester), code by night. I'm especially interested in **ethical and offensive security** — understanding how systems break so I can build them better.
